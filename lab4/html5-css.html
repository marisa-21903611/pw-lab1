HTML5 e CSS

HTML

HTML é a sigla de HyperText Markup Language, expressão inglesa que significa "Linguagem de Marcação de Hipertexto".

Consiste numa linguagem de marcação utilizada para produção de páginas de web, que permite a criação de documentos que podem ser lidos em praticamente qualquer tipo de computador e transmitidos pela internet.

Elementos HTML5
Elementos HTML Elementos utilizados Descrição
header  Este elemento indica o cabeçalho.
É utilizado em todos os cabeçalhos de todas as páginas do Web site.
nav  Este elemento indica o menu de navegação com os respetivos links das páginas do web site.
É utilizado em todas as páginas para representar o menu original do Web site, também foi utilizado em todas as páginas para representar o menu original do Web site.
main  Este elemento indica o conteúdo principal e único de cada página no site. É utilizado em todas as páginas de modo a apresentar o conteúdo que corresponde a cada uma.
article  Este elemento indica a porção do documento que pode ser vista de forma independente.
section  Este elemento especifica uma parte do conteúdo.
É utilizando etiquetas de cabeçalho, como do h1 ao h6. É utilizado em todas as páginas de modo a organizar melhor o código e colocando o conteúdos por subtítulos.
aside  Este elemento indica as anotações, a publicidade ou outros assuntos desconexos.
footer  Este elemento indica o rodapé do Web site.
É utilizado em todos os rodapés de todas as páginas do Web site, mostrando os dados do autor do trabalho, bem como a faculdade e o ano letivo do trabalho realizado.

details  Este elemento define detalhes adicionais que podem ser vistos ou serem omitidos.
É utilizado em todas as imagens que estão ao longo do site, tirando a imagem da cidade inicial que está inserida no header.
meter  Este elemento define medidas escalares estáticas dentro de um certo intervalo.
progress  Este elemento define uma barra de progresso de uma determinada tarefa.
abbr  Este elemento indica o significado de uma abreviatura.
Caso tenha este elemento basta colocar o rato em cima da palavra que este vai mostrar a palavra ou conjuntos de palavras que remete para a abreviatura.
address  Este elemento apresenta a informação sobre o contacto do autor do Web site.

CSS

O CSS, é a sigla de Cascading Style Sheets.

É uma "folha de estilo" composta por “camadas” e utilizada para definir a aparência em páginas da internet que adotem para o seu desenvolvimento linguagens de marcação (como XML, HTML e XHTML). O CSS define como serão exibidos os elementos contidos no código de uma página da internet, a sua maior vantagem é efetuar a separação entre o formato e o conteúdo de um documento.

Tipos de Seletores CSS
Tipos de Seletores Seletores utilizados Descrição
universal  Este elemento aplica o estilo a todos os elementos que pertencem à página.
É utilizado para definir algumas regras que serão implementadas no body.
tipo  Este elemento aplica-se ao nome do marcador a estilizar, faz com que seja aplicada a todos os elementos que possuam esse mesmo marcador.
É utilizado para definir o design do texto nas diferentes páginas, como a fonte do tipo de letra, a cor e o seu tamanho.
classe  Este elemento associa um nome a um estilo, faz com que todos os elementos que tenham uma certa palavra façam parte da regra que foi criada.
É utilizado em diferentes circunstâncias, tendo sido criado vários, para que fosse possível definir certas regras em diferentes tipos de texto.
Neste caso foi utilizado para definir a cor, a fonte do tipo de letra, o tamanho da mesma e para definir as imagens sobrepostas.
id  Este elemento indica permite identificar univocamente um elemento do HMTL, quando associado a uma palavra essa mesmo fica com a regra criada anteriormente.
É utilizado para definir regras únicas, neste caso a cor da borda, a cor do background e também o tipo de letra.
atributo  Este elemento combina elementos baseando-se no valor de um dado atributo.
É utilizado para definir as molduras das imagens, o tamanho da sua espessoura, a cor e o tamanho, mas também foi utilizado para os detlhes das imagens, para definir o tipo de letra.
pseudo-classe  Este elemento é considerado uma palavra-chave que quando adicionada a um seletor especifica um estado especial do elemento selecionado.
Este é utilizado para os links das diversas páginas, defenindo assim uma cor para cada processo até se abrir a página nova.
pseudo-elemento  Este elemento é também considerado uma palavra-chave que quando adicionada a um seletor permite que estilizar uma parte específica do elemento selecionado.
Este é utilizado para definir algumas regras, como por exemplo, cada parágrafo começar com letra maiúscula, bem como nas tabelas apresentadas ao longo do site estarem agrupadas em duas cores.

Combinadores de Seletores CSS
Combinadores Seletores Combinadores utilizados Descrição
descendentes  Este combinador, designado de "div p", faz com que sejam separados por espaço, o que acontece é que a regra se repete a cada espaço.
É utilizado em diversas páginas para colorir linhas do Web site.
filhos  Este combinador, designado de "div > p", faz com que todos os elementos(p) que pertençam ao "div" criado obedeçam à regra.
É utilizado em diversas páginas para colorir linhas do Web site.
adjacentes  Este combinador, designado de "div ~ p", faz com que todos os elementos(p) colocados após ao "div" criado obedeçam à regra.
É utilizado em diversas páginas para colorir linhas do Web site.
imediatamente adjacentes  Este combinador, designado de "div + p", faz com que todos os elementos(p) colocados "imediatamente" após os "div" criado obedeçam à regra.
É utilizado em diversas páginas para colorir linhas do Web site.
agrupados  Este combinador, designado de "div ,", faz com que as regras de todos os elementos(p) inseridos se cumpram.
É utilizado em diversas páginas para colorir linhas do Web site.
